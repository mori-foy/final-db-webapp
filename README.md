# いろにっき（Iro-Nikki）

色で一日を記録する、カレンダー型日記Webアプリです。

---

## 概要
本アプリは、日々の感情や出来事を言語化せず「色」で記録することを目的とした日記Webアプリです。  
記録をカレンダー形式で可視化することで、色の連なりから時間の流れや生活リズムを直感的に把握できるUIを設計しました。

---

## 主な機能
- 日記の作成・編集・削除（CRUD）
- 色による日記記録
- カレンダー形式での一覧表示
- 月切り替え（前月／次月）
- 今日の日付の強調表示

---

## UI設計の工夫
- 一覧表示をリストではなくカレンダー形式にすることで、記録の偏りや継続状況を視覚的に把握可能にしました。
- 色が記録されている日を浮き上がらせ、情報の有無が直感的に分かるデザインとしています。
- 今日の日付を枠線で強調し、現在地が分かりやすいUIとしました。
